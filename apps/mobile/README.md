# ProfitPlay Mobile

## Setup
- Install dependencies from the repo root: `npm install`
- Start the app: `npm run mobile:dev`

## Environment
Set the following variables when running locally or in EAS:
- `EXPO_PUBLIC_API_BASE_URL`
- `EXPO_PUBLIC_STRIPE_PUBLISHABLE_KEY`

## Builds
Use EAS build profiles in `eas.json` for dev, preview, and production builds.
